Intro

Throughout my apprenticeship in Thinkful's Data Analytics program, I've learned a great deal on advancing through the project life cycle regarding data. From conducting my own questional analysis to gathering data, then inturn cleaning that data to be more legible in the analysis process, and finally doing the nitty gritty calcualtions to find actionable insights that can be presentable to C-level executives, Thinkful has provided me with the experience needed to be able to work self sufficiently or cross-functionally in a working environment.
